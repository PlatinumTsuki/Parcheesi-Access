# Full keyboard reference — Parcheesi Access

## General navigation

| Key | Action |
|---|---|
| **Tab / Shift+Tab** | Cycle through interface zones |
| **Arrows** | Navigate the board cell by cell (never leaves the board) |
| **Enter** | Activate the focused button or element |
| **Escape** | Cancel / leave a view (context-dependent) |

## Main actions (in game)

| Key | Action |
|---|---|
| **Space** | Roll the dice |
| **1** to **4** | Select your piece + hear its position |
| **A** | Apply die 1 to the selected piece |
| **Z** | Apply die 2 to the selected piece |
| **S** | Apply the sum of both dice |
| **B** | Apply the bonus (capture +20, home arrival +10) |
| **T** | End the turn manually |

## Move previews

| Key | Action |
|---|---|
| **Shift+A** | Preview die 1 (where the selected piece would land) |
| **Shift+Z** | Preview die 2 |
| **Shift+S** | Preview the sum |
| **Shift+B** | Preview the bonus |

## On-demand announcements

| Key | What |
|---|---|
| **D** | Current dice state (values, which are used) |
| **I** | Full list of every legal move for the selected piece |
| **L** | Read the entire board (every player's pieces) |
| **J** | Opponent pieces only |
| **R** | Repeat the last announcement word for word |
| **P** | Last 5 lines of the game journal |
| **K** | Cumulative statistics (by difficulty, by personality) |
| **M** | Full achievement list (unlocked + locked) |
| **C** | Tactical advice (move recommended by Hard-level AI) |
| **H** | Full help (reminder of every key) |

## Pause and timer

| Key | Action |
|---|---|
| **F2** | Pause / resume |
| **F3** | Announce remaining time for this turn (if timer is enabled) |

## Tutorial

| Key | Action |
|---|---|
| **Space** / **Enter** / **→** | Advance to the next step |
| **Escape** | Exit the tutorial |
| **Any other key** | Exit the tutorial and trigger that key's normal action |
